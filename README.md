# Fake Productivity Detector & Focus Analyzer

## Overview
This project analyzes user activity and calculates a productivity score based on time spent on productive and distracting applications.

## Features
- Productivity Score Calculation
- Focus Analysis
- Distraction Detection
- CSV Data Processing

## Technologies Used
- Python
- Pandas
- CSV Files

## Project Structure

Fake-Productivity-Detector
├── activity.csv
├── productivity_analyzer.py
└── README.md

## Sample Data

| App Name | Time (Minutes) | Category |
|-----------|---------------|----------|
| VS Code | 120 | Productive |
| Word | 60 | Productive |
| YouTube | 90 | Distracting |

## Output

Productivity Score: 52.94%

Status: Moderately Productive
